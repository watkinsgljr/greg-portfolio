Employee Data Management System:
I got a chance to build an Employee Data Management application that collects user input and stores the information into its database. The app then fetches the data, manipulates it, and imports it into the jQuery data table.

This was a fun project because I challenged myself to finish it in one day. I didn't get as far as I would have liked but it was a great learning experience, particularly with data tables. I got stuck when I realized that users would not have sorting capabilities with the original table's data due to it being only an HTML markup instead of an actual data storage medium where users could interact with the information.

Spotlight experience:
Table vs data table. jQuery data table Firebase.

Other experience/tech used:
HTML, CSS, Javascript, accounting.js library, moment.js library, and Bootstrap Datepicker.

Future development:
Design, allow users to edit and remove employees from app, add modals that notify user when new data has been submitted successfully, and sort icon so that user knows the columns can be sorted.

MGM Grand Street:
This boxing game allows user to select one of four boxers in which they will battle for the championship. Each boxer has a different punch power, counter punch power, and beginning health points to start the game so users must select the order they fight each opponent carefully.

On each punch (click of attack button): User's boxer's punch power increases by the amount of starting punch power. Opponent counter punches user. User health points decrease by opponent's counter punch power. Opponent health points decrease by user punch power. Heath bar resized to reflect percentage of health remaining for each user.

Spotlight experience:
Great practice with algorithms.

Other experience/tech used:
JavaScript, jQuery, HTML, CSS.

Future development:
Add flip feature to boxing cards so that boxer information is on the back of the card and is displayed on hover.

GiphEmotions Giphy Generator:
This Giphy Generator uses Giphy API to specialize in emotion gif generation. Click on an emotion button and 8 gifs populate that correspond with the button clicked.
